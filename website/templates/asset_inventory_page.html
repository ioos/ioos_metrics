Field Definitions

Field Definition
RA Name of the Regional Association
station_long_name Station name. e.g. Red Bank - Garden Banks, Fripps Inlet, etc.
latitude Coordinate Reference System WGS84 (EPSG:4326).
longitude Coordinate Reference System WGS84 (EPSG:4326).
Platform A structure or vehicle designed to hold one or more sensors such that the intended oceanographic or atmospheric variable(s) can be monitored or measured in a manner designed for the ocean observing system or scientific hypotheses.
station_deployment The time when the station became operational. This can be determined by the first data record or an installation date. This indicates the first time the station was deployed - not the last or latest, but the first time - even if IOOS was not established yet.
Operational Is the asset deployed and collecting data?
Y = Yes (this includes seasonal buoys intended for redeployment).
N = No (it was uninstalled/removed, or is offline with no intent to repair).
O = Offline (temporary outage, with intent to repair or redeploy).
U = Unknown (e.g. funding uncertainty for repair/redeployment, or operator's plans are unknown).
RA_Funded Does the RA currently fund the asset? This excludes support for data management activities.
Yf = Fully funds.
Yp = Partially funds.
N = No funding.
Raw_Vars A list of observed variables that are associated with each station, with the depth(s) of the each measured oceanographic variable in parentheses following the variable name.
